# ScreenCap Studio

[![Live Demo](https://img.shields.io/badge/Live_Demo-screencap--studio.surge.sh-brightgreen?style=for-the-badge)](https://screencap-studio.surge.sh)

> Free browser-based screenshot annotator. No signup, no upload, works offline.

**Try it now:** [screencap-studio.surge.sh](https://screencap-studio.surge.sh)

## Features

- **Arrow** — Point to anything with styled arrows
- **Rectangle** — Draw boxes around areas of interest
- **Circle / Ellipse** — Highlight regions
- **Text** — Add labels and notes
- **Numbered markers** — ①②③ step-by-step guides
- **Blur** — Redact sensitive info (passwords, tokens, faces)
- **Highlight** — Semi-transparent color overlay
- **Undo / Redo** — Full edit history
- **Export PNG** — Original resolution output
- **Copy to clipboard** — One-click sharing
- **Paste from clipboard** — Ctrl+V to start

## Why?

Every day millions of people share annotated screenshots on Slack, Twitter, GitHub, and docs. Current tools (Snagit, CleanShot X) are paid desktop apps. This is free and runs in any browser.

## Tech

Single HTML file. Zero dependencies. Pure Canvas API.

- Canvas 2D for all drawing operations
- Clipboard API for paste/copy
- Drag & Drop API for file loading
- Keyboard shortcuts for power users
- Export at original image resolution

## Quick Start

```
Just open index.html in a browser. That's it.
```
